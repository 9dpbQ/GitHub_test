どんなカスタマイズができるのか What Can You Customize?

⌨️

キーのリマップ Key Remapping

好きなキーを好きなキーに割り当て直せます。 Remap any key to any other key.

🗂️

レイヤー機能 Layer Function

Fnキーなどで別の配列に切り替え可能。 Switch to another layout with Fn key, etc.

🤝

コンボ Combo

複数キー同時押しで新しいアクション。 Trigger new actions with key combos.
